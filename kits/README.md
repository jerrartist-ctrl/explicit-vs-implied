# Kits (Public Previews)

This folder contains **public preview versions** of my kits—lightweight samples meant to show structure, scope, and clarity.

Each kit folder typically includes:
- `README.md` — overview + what it’s for
- `product-page.md` — storefront-style description / positioning
- `assets/` — images used by the pages
- `LICENSE` — usage terms for the preview content
